# Devano — Gerador de Proposta

## Deploy no Vercel (passo a passo)

1. Vá em **github.com** → New repository → `devano-proposta` → Create
2. Na página do repositório, clique **uploading an existing file**
3. Arraste **todos os arquivos desta pasta** para o upload (inclua a pasta `src`)
4. Clique **Commit changes**
5. Vá em **vercel.com** → Add New → Project
6. Selecione o repositório `devano-proposta`
7. Framework: **Vite** (detectado automaticamente)
8. Clique **Deploy**

Pronto! Em ~2 minutos você terá um link como `devano-proposta.vercel.app`
